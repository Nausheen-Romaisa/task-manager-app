Task Manager App

Managing daily tasks can get overwhelming, especially when things start piling up.  
This Task Manager App is a simple yet effective solution designed to help users stay organized, focused, and productive.

Built using core web technologies, this project reflects my understanding of frontend development and my ability to create interactive, user-friendly applications.

What This Project Does

This application allows users to:

- Add tasks quickly and efficiently  
- Remove completed or unnecessary tasks  
- View tasks instantly with real-time updates  
- Interact with a clean and responsive interface  

The goal of this project is to provide a minimal and distraction-free environment for managing everyday tasks.

Why I Built This

As part of my learning journey in web development, I wanted to build a project that is:

- Practical and useful in real life  
- Focused on user experience  
- Simple, but functionally complete  

This project helped me strengthen my understanding of DOM manipulation, event handling, and UI design.

Tech Stack

- HTML → Structure of the application  
- CSS → Styling and layout  
- JavaScript → Functionality and interactivity

How It Works

1. User enters a task in the input field  
2. Task is dynamically added to the list  
3. Each task has an option to delete  
4. Changes reflect instantly without reloading
